'Directive' Operands

Several operands can be used throughout BASM programs to interact with a user, or provide extra functionality relating to text and console manipulation.

  • %0

    • Explanation: Reads 1 key from standard input.
    Example: PNT %0
    The code above reads 1 key from the standard input and prints it out.
    Output: aa
    The key that was received as input is printed, then printed again.

  • %1

    • Explanation: Reads standard input until return is pressed.
    Example:
    PNT "Enter your name: "
    MOV R0, %1
    PNT "Hello "
    PNT R0
    Output:
    Enter your name: Ben _
    Hello Ben

  • /n

    • Explanation: These two characters, present anywhere in a string, will be treated as a line feed at runtime.
    Example:
    PNT "This is the first line!/nThis text is presented below the first line!"
    Output:
    This is the first line! This text is presented below the first line!
