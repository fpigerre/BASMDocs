Introduction

BASM, also known as "Ben's Assembly", is a simplified Assembly language run using a virtual processor. BASM follows conventions similar to the Intel x86's Assembly language in order to provide an easy and stress-free learning curve. Aimed at being a portable Assembly language, one of BASM's main goals is to be able to be run on as many different architectures as possible. BASM simplifies the referencing of registers by using lettering to determine storage space, instead of hexadecimal numbers.
